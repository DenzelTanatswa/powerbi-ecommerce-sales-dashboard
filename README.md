# E-Commerce Sales Dashboard in Power BI

## Project Overview
Developed an interactive Power BI dashboard to analyze 50,000 e-commerce transactions and generate actionable business insights.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Data Visualization
- Business Intelligence

## DAX Measures
- Total Revenue
- Total Transactions
- Average Purchase
- Highest Purchase

## Key Findings
- Total Revenue: R25.16M
- Total Transactions: 50,000
- Average Purchase: R503.16
- Highest Purchase: R999.98
- Highest Revenue Category: Sports
- Most Purchased Category: Toys
- Preferred Payment Method: UPI
- Highest Revenue Country: France
